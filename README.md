## 선행 작업
* json-server 전역 설치
```npm install -g json-server```

## 설치 및 실행 방법
* 설치: ```npm install or yarn```
* json-server 실행: ``` json-server --watch ./db/shop-item.json --port 3001```
* application 실행: ```npm run start or yarn start```
